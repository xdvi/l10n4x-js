# @l10n4x/wasm

Thin npm alias for the WebAssembly bindings built from `l10n4x-wasm` in the main [l10n4x](https://github.com/xdvi/l10n4x) repository.

## Install

```bash
npm install @l10n4x/wasm l10n4x-wasm
```

For local development against a sibling `l10n4x` checkout, run `pnpm link-wasm` from the monorepo root.
